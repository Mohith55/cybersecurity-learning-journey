# Day 03 â€“ OSINT with WHOIS

## Objective
Investigate a suspicious crypto investment website using OSINT techniques.

## Method
- Performed WHOIS lookup on the target domain
- Analyzed registrar, hosting provider, and registration timeline

## Findings
- Recently registered domain
- Short expiration period
- Shared hosting with suspicious domains
- Masked or unreliable contact details

## Key Insight
WHOIS is a foundational OSINT tool that helps:
- Identify scam infrastructure
- Link multiple fraudulent domains
- Preserve evidence for reporting

## Takeaway
Investigation comes before exploitation.
Understanding infrastructure is critical in cybersecurity.
